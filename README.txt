Electronic notebook for "Remarkably high and consistent tolerance of a Red Sea coral to rapid and prolonged thermal stress"

Open Access publication accessible here: https://aslopubs.onlinelibrary.wiley.com/doi/full/10.1002/lno.11715 

Contains two folders: one with all the main figures and one with the supplementary figures

Each Folder contains a PDF of the figure, the data and R code used to produce the figure, and the R code detailing how the statistical analyses were conducted.

Note that figures 3, 4, and S2 were produced with Prism by Graphpad (v8) using the provided data.
